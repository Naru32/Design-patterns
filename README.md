# React Design Patterns

This repository contains practical implementations of important React design patterns.

## Patterns Covered

### 1. Single Responsibility Principle (SRP)
- Debouncing logic using custom hook
- Separation of logic and UI

### 2. Container vs Presentational
- Separation of data logic and UI components

## How to Run

Go inside any folder:

cd Single-Responsiblity 
npm install  
npm run dev
